# Web Project

## Online Learning Platform: ULearn

- Clone this repo:

  ```shell
  git clone -b develop https://github.com/duclm278/web-project.git
  cd web-project
  git checkout -b <your-name> # git checkout -b duc
  ```

- Install dependencies:

  ```shell
  # From project, cd to frontend:
  cd frontend
  npm install
  ```

  ```shell
  # From project, cd to frontend:
  cd backend
  npm install
  ```

- Run backend, frontend and admin-panel:

  ```shell
  cd backend
  npm install
  npm run dev
  ```

  ```shell
  cd frontend
  npm install
  npm run dev
  ```

  ```shell
  cd admin-panel
  npm install
  npm run dev
  ```

- Work with Git:

   ```shell
    git add . # Add changes
    git commit -m "<your-message>" # Commit changes
    git push -u origin <your-branch> # git push -u origin duc
    ```
